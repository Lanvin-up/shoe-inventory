📖 GitHub Gist 免费使用说明
1. 登录 GitHub，访问 gist.github.com 新建一个私密代码片段（内容可留空）。
2. 复制浏览器地址栏中的 Gist ID（例如 https://gist.github.com/你的用户名/1a2b3c4d5e6f → ID 为 "1a2b3c4d5e6f"）。
3. 前往 GitHub Token 设置页，勾选 gist 权限，生成 Token 并复制保存。
4. 将 Token 和 Gist ID 填入上方，点击「备份到云端」上传当前库存。
5. 「从云端恢复」将拉取最新备份并覆盖本地数据。
✅ 免费且无流量限制，个人使用完全足够。
⚠️ Token 仅保存在浏览器本地，请勿泄露。
